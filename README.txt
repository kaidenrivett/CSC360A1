NOTE: Before running PMan, make sure to compile the included source files with gcc. I would include the binary files as well,
however, when I was testing on the UVic machines the permissions would be denied. Sorry for the inconvenience.

Running PMan:
1. make
2. ./PMan


Accepted Commands:
- bg [args]
- bglist
- bgkill [pid]
- bgstop [pid]
- bgstart [pid]
- pstat [pid]
- exit


Included Files:
- makefile        ->      to compile the PMan program
- PMan.c          ->      PMan Program
- args.c          ->      program that prints out the arguments passed
- inf.c           ->      program that accepts a tag and an interval
- constoutput.c   ->      program that prints out constant output 


How to exit PMan:
- To exit PMan type in the command "exit"
